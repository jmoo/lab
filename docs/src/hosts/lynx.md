- **Platform:** NixOS, `x86_64-linux`.
- **GPU:** NVIDIA (`hyprland.nvidia = true` in `nixos.home`); `boot.binfmt`
  emulates `aarch64-linux`.
- **Remote builder:** serves `nix-ssh` (`nix.sshServe`) so meerkat can offload
  builds here; signing/SSH pubkeys live in `keys/`.
- **Services:** jellyfin, sunshine, steam, wireshark, pipewire, printing.
- **Tailscale:** exit node / subnet router (`tailscale = { enable = true; exitNode = true; }`).

Defined in `hosts/lynx.nix` (hardware inlined into `nixos.module`). Rebuild with
`sudo nixos-rebuild switch --flake .#lynx`.
